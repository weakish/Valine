Avatar Setting

Edit On Github

Valine is used in Gravatar (with gravatar.cat.net mirror service) as commentator avatar.
Has seven optional values:

value style memo
Empty string:'' Gravatar official graphics
mp Mystic man (a grayhead)
identicon Abstract geometry
monsterid little monster
wavatar A combination of different faces and backgrounds
robohash a generated robot with different colors, faces, etc
retro Eight-pixel retro portrait
hide   Hidden avatar

The parameter value hide is supported at v1.1.8-beta.
If the value of avatar is not equal to one of the seven parameters, the default value ismm.

new Valine({
    ...
    avatar:'' // (''/mm/identicon/monsterid/wavatar/robohash/retro/hide)
});
